# Learn Claude Workspace

This workspace is organized for learning and creating projects with Claude.

## Structure

- `.github/`: Contains customization files for VS Code agents, including skills.
  - `skills/`: Folder for custom skills. Each skill should be in its own subfolder with a `SKILL.md` file.
- `projects/`: Folder for categorized projects.
  - `web-development/`: Projects related to web development.
  - `data-science/`: Projects related to data science.
  - `ai-ml/`: Projects related to AI and machine learning.
  - `other/`: Other miscellaneous projects.

## Getting Started

To create a new skill, create a new folder under `.github/skills/` and add a `SKILL.md` file following the skill template.

To start a new project, create a subfolder under the appropriate category in `projects/`.

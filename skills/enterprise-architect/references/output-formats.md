# Output Formats

Use the appropriate format template for each task type.

---

## ADR Format

```
# ADR-[NUMBER]: [Title]
**Date**: [Date]  
**Status**: [Proposed | Accepted | Deprecated | Superseded]  
**Deciders**: [Stakeholders]

## Context
## Decision
## Consequences
### Positive
### Negative
### Risks
## Alternatives Considered
## References
```

---

## Architecture Review Format

```
## Executive Summary
## Architecture Overview
## Findings
### Critical Risks
### High Risks
### Medium Risks
### Low Risks / Observations
## Technical Debt
## Recommendations

```

## Gap Analysis Format

```
## Current State Summary
## Target State Summary
## Gap Analysis
| Capability | Current State | Target State | Gap | Priority |
## Migration Considerations
## Recommended Roadmap
```

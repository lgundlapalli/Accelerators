# PRD Writer — Samples

Best-in-class PRD examples. Used as quality benchmarks when running the anti-slop checklist. Add examples that demonstrate the standard to meet or exceed.

## Folder Convention

```
samples/
  [ProductName]-PRD-[YYYY-MM-DD].md    (markdown version)
  [ProductName]-PRD-[YYYY-MM-DD].docx  (Word version)
```

## Grading Criteria for a Sample to Qualify

A PRD belongs here only if it passes all of the following:

- [ ] Hypothesis uses if/then/because format with a specific number
- [ ] Every metric has a threshold and a baseline
- [ ] Kill criteria names a specific rollback trigger
- [ ] AI Behavior Contract has 10+ labeled GOOD/BAD/REJECT examples
- [ ] Tradeoffs Accepted names something real
- [ ] No banned phrases present
- [ ] Dependencies have named owners
- [ ] Graduate When criteria is specific and measurable
